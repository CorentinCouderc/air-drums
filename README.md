# air-drums
Projet de 96h de fin de cursus CPE Lyon

## Auteurs :
- COUDERC Corentin
- MIOR Edouard
- LEPLOMB Théo

Ce projet marche avec Kinect v1 et OpenNI + Sensor.

## Compiler : 
```
cd air-drums
sh compilation.sh
```
## Executer :
```
cd air-drums
sh exe.sh
```
